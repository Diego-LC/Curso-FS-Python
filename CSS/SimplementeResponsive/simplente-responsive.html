@media queries

CSS uses the @media rule to detect information about the device being used to acces the website. for example, it can be used to detect the size of device, its capabilities, and its orientation For this course, we will be using it to detect the size, but keep in mind that it is capable of much more

Desktop: Width greater than or equal to 1024px

Desktop: Width between 481px and 1023px

Desktop: Width less than or equal to 480px
